Shared Shop Application

ECサイトの基本機能を実装したSpring Boot学習用アプリケーション。

GitHub 画面一覧

主な機能

📁 商品管理

管理者が商品の登録・更新・削除や画像管理を実施可能

🔍 商品一覧・検索

カテゴリ検索や並び替えに対応し、新着順・売れ筋順で商品を表示できる。

🛒 買い物かご

商品追加や数量変更に対応し、在庫状況を確認しながら管理できる。

📦 注文フロー

配送先入力から注文確定までを4ステップで進める購入機能。

📒 注文履歴・管理

注文履歴の確認に対応し、管理者は全注文を管理できる。

👤 会員管理

会員情報の登録・編集に対応し、管理者は会員管理を行える。

🔐 ログイン管理

ログイン失敗回数を記録し、5回連続で失敗すると30分間ロックする。

💬 お問い合わせ

アンケート形式で管理者へ意見を送信し、内容を管理画面で確認できる。

⭐ レビュー

アバターがレビューを表示し、メリット・デメリットや星評価を確認できる。

🎁 ポイント機能

購入金額の1％をポイント付与し、商品や福袋の購入に利用できる。

⚠️ かご落ち通知

カート内の商品を一定時間放置すると通知を表示して購入を促す。

🎂 誕生日祝福

誕生日当日に祝福ポップアップを表示し、特別な演出を行う。

🎊 福袋機能

価格別の福袋を販売し、商品をランダムに組み合わせて提供する。

🛡️ アクセス権限管理

管理者と一般顧客の権限を分離し、機能や画面へのアクセスを制御する。

🔮 占い・おすすめ

占い結果に応じた商品を表示し、当日限定の割引販売を行う。
